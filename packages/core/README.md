# @knexbridge/core

Core library for KnexBridge. It provides database introspection utilities and code generation helpers that emit TypeScript interfaces, insert/update helpers, and Zod schemas.

## Installation
`ash
npm install @knexbridge/core
`

## Usage
`	ypescript
import { introspectDatabase, generate } from '@knexbridge/core';
`

Refer to the main KnexBridge documentation for end-to-end examples.
